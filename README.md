
# VAI Challenge

Vai challenge api

## Installing / Getting started

```shell
npm install
npm start
```

### Built With
Express framework

### Prerequisites
Node.js version 8.0.0 and upper 
MongoDB

### Setting up Dev

Clone project:

```shell
git clone https://github.com/edwardntn/challenge.git
cd vai-challenge/
npm install
```

Set db url, port and host in `/config/index.js`

## Configuration

server port -  
PORT

database settings -  
DB_URI

## Style guide

Run script to check code style:

```shell
npm run lint
```

## Database

The database used is MongoDB.
